# Font types

* Stencil is only used to render the 1842 year in the logo on the splash screen.
* MaterialDesign come an open library of icons https://pictogrammers.github.io/@mdi/font/7.2.96/
* OpenSans is used for all other text in the app.

# Building

In order to ligthen the app opening, we need to restrict the fonts to only the needed icons, especially for MaterialDesign. We've been using
FontForge to remove what is not needed. This implies that adding any new icons, will need to be added manually.
